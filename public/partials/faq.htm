Frequently Asked Questions

Can I download the repository?

It is currently not possible to download an anonymized repository. It is technically possible to implement but would require much more processing and storage. I am currently not able to provide this option due to the increase cost.

Which file formats are supported?

How much is cost?

Anonymous Github is free to use, however the server costs me hundreds of dollars per year, a small donation to cover to cost is largely appricated.

Data privacy?

The data stored on Anonymous Github are never used or shared in any cases. When a repository is removed or expired only the configuration of the repository is conserved to be able to restore easily the reposiotry and to ensure that no future repository will use the same repository id.

Privacy of the viewer?

No information is collected on the viewer of the repositories.
